# repository_git-laba
#Название проекта: Игра "Угадай число"

#Описание:
Игра "Угадай число" — это простая текстовая игра на C++, в которой игрок пытается угадать случайное число, загаданное программой.
Игрок вводит числа в заданном диапазоне, и программа сообщает, больше или меньше загаданное число относительно введённого. 
Игра продолжается до тех пор, пока игрок не угадает число.

#Инструкции по запуску.
#Необходимые инструменты:
1.Компилятор C++: Для компиляции и выполнения кода вам потребуется компилятор C++.
2.Текстовый редактор или IDE.
